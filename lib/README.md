TITLE is a theme for [mdcss].

## Usage

Add [mdcss] and [TITLE] to your build tool:

```bash
npm install mdcss --save-dev
npm install THEME --save-dev
```

Whenever [mdcss] is used, reference this theme.

```js
require('mdcss')({
	theme: require('THEME')({ /* options */ })
})
```
